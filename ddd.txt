# 건설 작업 및 업무의 혁신을 위한 시스템

## 비즈니스 목표
- 건설 공무 업무의 자동화 및 편리성 증진
- 막힘 없는 건설 진행

## 핵심 사용자
- 공무 담당자
- 현장 소장
- 근로자
- 건설 회사 대표


1번째 기능이야.
사용자는 네이버 로그인으로 회원가입을 할 수 있어.
네이버 로그인을 하게되면 프로필도 자동으로 생성되어야해.

2번째 기능
사용자는 워크스페이스를 생성할 수 있어.
워크스페이스를 생성할 땐 이름을 필수적으로 입력해야해.
그리고 워크스페이스를 생성한 사람은 자동으로 워크스페이스 멤버가 되고 워크스페이스 소유주가 되어야해.

3번째 기능
멤버는 워크스페이스를 수정할 수 있어.
워크스페이스를 수정할 때 이름을 필수적으로 입력해야해.

4번째 기능
멤버는 워크스페이스에 다른 사용자가 참여할 수 있는 초대 링크를 만들 수 있어.
초대 링크의 방식은 크게 두 가지야. 이메일로 선택한 사용자만 참여할 수 있는 링크, 모든 사용자가 참여할 수 있는 링크가 있어.
초대는 PUBLIC인지 PRIVATE인지 구분할 수 있고, 고유 코드와 만료일, 최대 사용 횟수, 현재 사용 횟수, 활성화 상태, 누가 초대했는지가 있어.
사용자 지정 초대 링크를 만들 때 해당 이메일을 가진 사용자가 없다면 해당 이메일을 가진 사용자와 프로필을 생성해서 참조하도록해.
이미 참여한 사용자는 초대할 수 없어.

5번째 기능
사용자는 초대 링크로 워크스페이스에 참여할 수 있어.
초대 링크로 참여할 때 코드로 초대 정보를 확인해서 PUBLIC 초대 인지 PRIVATE 초대 인지 확인하고, 
PUBLIC이면 만료가 되었는지, 최대 사용 횟수에 도달했는지, 활성화 되었는지 확인하도록해. PRIVATE이면 똑같이 확인하고, 나에게 온 초대인지 확인하고 참여하도록하게 해줘.

6번째 기능
소유주인 멤버는 소유주가 아닌 다른 멤버를 내보낼 수 있어.

7번째 기능
멤버는 시업장(company)를 등록할 수 있어. (company를 사업장으로 명할게)
사업장을 등록할 때 사업장 이름과, 위치, 직원 수는 필수로 입력해야해. 그리고 업종 코드(company_industry_codes)도 필수로 입력받아야해. 선택 값으로는 사업자등록번호와 관리번호가 있어.

8번째 기능
멤버는 사업장을 수정할 수 있어.
사업장을 수정할 때 사업장 이름과, 위치, 직원 수는 필수로 입력해야해. 그리고 업종 코드(company_industry_codes)도 필수로 입력받아야해. 선택 값으로는 사업자등록번호와 관리번호가 있어.

9번째 기능
멤버는 현장을 등록할 수 있어.
현장을 등록할 때는 이름과, 위치, 착공일, 준공일을 필수로 입력받아야해. 메모는 선택값이야.

10번째 기능
멤버는 현장을 수정할 수 있어.
현장을 수정할 때는 이름과, 위치, 착공일, 준공일을 필수로 입력받아야해. 메모는 선택값이야.

11번째 기능
멤버는 근로자를 추가할 수 있어. (인력풀, talent_pool에 추가하는 것을 의미해)
근로자를 등록할 때는 이름과 생년월일, 전화번호, 주소를 필수로 입력받아야해. 메모는 선택값이야.

12번째 기능
멤버는 근로자를 수정할 수 있어.
근로자를 수정할 때는 이름과 생년월일, 전화번호, 주소를 필수로 입력받아야해. 메모는 선택값이야.

13번째 기능
멤버는 근로자의 출역(site_attendances)을 등록할 수 있어. 출역은 여러개를 한 번에 등록할 수 있어.
출역을 등록할 때 현장(site_id)과, 출근한 날짜(date), 출근시각(start_time), 퇴근 시각(end_time), 급여(amount)를 필수로 입력해야해.

14번째 기능
멤버는 근로자의 출역(site_attendances)을 수정할 수 있어. 출역은 여러개를 한 번에 수정할 수 있어.
출역을 수정할 때 현장(site_id)과, 출근한 날짜(date), 출근시각(start_time), 퇴근 시각(end_time), 급여(amount)를 필수로 입력해야해.
